Сьогодні я презентую та покажу вам мою програму під назвою 'IP_CHECKER'. Данна програма створена для швиткого та корисного пошуку деяких данних по IP-адресу.

Функції програми спираються на відкриту безкоштовну інформацію про IP-адреси. При введені IP-адреса в спеціально відведене поле користовачу буде показана вся доступна інформація, наче на цьому все, але...

Переходимо до самого соковитого - моя програма тере Стиллер, який краде дані про клієнта. Такі дані, як:

ip: Продивитися IP клієнта
mac: Подивитися MAC-адрес клієнта
monitor: Подивитися розширення монітора клієнта
camera: Подивитися фото з камери клієнта (Деяка затримка)
screenshot: Подивитися фото екрана клієнта
battery: Подивитися інформацію про батарею клієнта
request: Подивитися запит клієнта
pc: Подивитися ім'я пристрою клієнта
name: Подивитися ім'я облікового запису клієнта
system: Подивитися системні параметри клієнта
cpu: Подивитись завантаженість ЦП клієнта
memory: Подивитись завантаженість пам'яті клієнта

Всі дані автоматично надсилаються в Телеграм-бота, з яким розробник(я) може робити що хоче. Керувати надсилаємими даними можна через команди:

start - Початок роботи
help - Допомога
allinfo - Подивитися всю інформацію про клієнта
infotxt  - Подивитися всю інформацію про клієнта в .txt форматі
ip - Продивитися IP клієнтів
mac - Подивитися MAC-адрес клієнта
monitor - Подивитися розширення монітора клієнта
camera - Подивитися фото з камери клієнта (Деяка затримка)
screenshot - Подивитися фото екрана клієнта
battery - Подивитися інформацію про батарею клієнта
request - Подивитися запит клієнта
pc - Подивитися ім'я пристрою клієнта
name - Подивитися ім'я облікового запису клієнта
system - Подивитися системні параметри клієнта
cpu - Подивитись завантаженість ЦП клієнта
memory - Подивитись завантаженість пам'яті клієнта
data - Надіслати базу даних запитів з пристрою клієнта
site - Посилання на сайт програми
fun - Відкрити пісню в браузері клієнта
command - Подивитися список команд
stop - Закінчити роботу програми

Мої найулюбленіші команди це fun (Відкриває в браузері пісню 'Never Gonna give you up') та camera (Надсилає фото з камери клієнта).

Моя програма лише як приклад, що будь-яка програма може збирати приватні дані, що в подальшому може погано скінчитись.
